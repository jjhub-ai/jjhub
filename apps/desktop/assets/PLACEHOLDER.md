Place the following icon files in this directory:

- icon.png      -- 1024x1024 app icon (used for macOS, Windows, Linux)
- icon.icns     -- macOS icon set (generated from icon.png)
- icon.ico      -- Windows icon (generated from icon.png)
- tray.png      -- 22x22 system tray icon (template image, single color)
- tray@2x.png   -- 44x44 retina tray icon

You can generate platform-specific icons from a single 1024x1024 PNG using
tools like `electron-icon-maker` or `iconutil` (macOS).
